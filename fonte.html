De Onde Vêm os Dados?

Origem e Fontes de Dados

Os conjuntos de dados foram obtidos na plataforma Kaggle, uma comunidade de cientistas. Utilizamos datasets extraídos por web scraping que incluem:

- Dados de crimes fornecidos pela Secretaria de Segurança Pública (SSP), referente ao ano de tal.

- Dados de imóveis da plataforma Quinto Andar, referentes ao ano de tal.

Segurança da Informação

Todos os dados são anonimizados, sem informações pessoais ou sensíveis, garantindo a privacidade e a segurança dos usuários.

Tratamento e Análise de Dados

A preparação e análise dos dados foram realizadas utilizando Python, com o auxílio das bibliotecas Pandas, Geopandas, Matplotlib e Folium. A ferramenta Power BI também foi empregada para proporcionar uma filtragem mais precisa e visualização de dados eficiente.
